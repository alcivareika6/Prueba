Especificidad

La especificidad es una unidad de medida para determinar cómo de específico es el selector de una norma de estilo. Tiene cuatro componentes que, de menor a mayor, son:

  1. Etiquetas HTML
  2. Clases y pseudoclases
  3. IDs
  4. Estilos en línea (style)
