 # Convenciones de escritura de código

Estas convenciones hacen la diferencia entre un código profesional y uno _aficionado_.

## 1. lowerCamelCase
Es el estándar más utilizado en JavaScript.

**Usos:**

- Variables: `let edadPersona = 25;`
- Funciones: `function calcularEdad () {...}`
- Métodos: `obj.devolverEdadPersona ()`
- Propiedades de los objetos: `edadPersona: "32"`

## 2. UpperCamelCase (PascalCase)

**Usos**:
- Clases:`class AlumnoCurso {...}`  
- Interfaces ...

## 3. SCREAMING_SNAKE_CASE

**Usos:**
- Constante dew JavaScript ` const NUMERO_PI = 3.1416;`

## 4. kebab-case

**Usos:**
